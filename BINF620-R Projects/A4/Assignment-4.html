Missing data is defined as the values or data that is not stored/not present for some variable/s in the given dataset.
- Identify types of Missing Values - Brief describe the methods to handle missing values in a dataset

There are three types of missing values. MCAR, MAR, and NMAR. - MCAR: Missing Completely at Random. These missing values occur in no pattern and have no relation to any of the other values of a variable. This is often the rarest type of missing value. - MAR: Missing at Random. This is a little different being, can be further explained by already known or obtained data. - NAMR: Not missing at Random. The conclusion can be made if the missing values fit neither of the other discussed categories. Implying, there there could be a reason the values are missing or the value is missing because another value provides a reason for a missing value.
